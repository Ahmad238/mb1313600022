### Fruitin App
Fruitin is an application that displays the nutrition of various fruits. This application will help users to find some of the nutritional content in the fruit.

### Target Segment
* Users in general
* Vegetarian users
* Users who have a diet program

### Advance Topic
* REST API

### Features
* List of fruits
* Liked
* Bookmark
* Search
